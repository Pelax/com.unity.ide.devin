# Devin / Windsurf Editor Integration for Unity

A Unity package that adds support for **Devin Desktop** (formerly **Windsurf**) as an external code editor in Unity. This package supports auto-discovery of installations, generating `.csproj` and `.sln` files for IntelliSense (via SDK-style project files), and setting up debugging workspace configurations.

## How to Install

1. Open Unity.
2. Go to **Window** -> **Package Manager**.
3. Click the **"+"** button in the top-left corner.
4. Select **"Add package from git URL..."**.
5. Insert the git repository URL:
   `https://github.com/Asuta/com.unity.ide.windsurf.git`
6. Click **Add**.
7. Go to **Edit** -> **Preferences** -> **External Tools** and select **Devin Desktop** or **Windsurf** from the **External Script Editor** dropdown list.
